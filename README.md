# Админ-панель Пеликан Алаколь

Простая панель управления контентом для приложения.

## Быстрый старт

1. **Настройте config.js** - укажите ваш репозиторий
2. **Откройте index.html** в браузере
3. **Настройте GitHub токен** - нажмите Ctrl+K

## GitHub токен

1. Создайте Personal Access Token в GitHub:
   - Settings → Developer settings → Personal access tokens
   - Выберите scope: `repo` (полный доступ к репозиториям)

2. В админ-панели нажмите Ctrl+K и введите токен
